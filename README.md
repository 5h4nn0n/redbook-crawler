# redbook-crawler

A Chrome extension to extract notes data from Redbook (Xiaohongshu) website.

## Features

- Extract notes data from Redbook website.
- Support extract notes detail from explore page.
- Support extract user profile from user profile page.
- Support batch extract user profile from discovery page.
- Export data as JSON file.

## Installation

1. Download the source code.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" on the top right corner.
4. Click "Load unpacked" button on the top left corner.
5. Select the folder where you download the source code.
6. The extension should be installed now. You can see it on the extensions list.
7. Click the extension icon on the top right corner to open the popup.
8. Grant the necessary permissions when prompted.
9. Please use it responsibly and do not use it for illegal purposes.
10. The extension is still under development. Some features may not work as expected.
11. If you have any suggestions or feedback, please feel free to open an issue or pull request.
12. Thank you for using this extension!
13. Enjoy the extension!
